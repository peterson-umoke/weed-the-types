== Description ==

This is a plugin that is used to delete unwanted post types and product types. It simply provides a menu for people to select which post type contents they want to get rid.

== Installation ==

1. Upload `weed-the-types.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
